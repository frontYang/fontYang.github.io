Hexo常用命令备忘

发表于 2018-03-04 | 分类于 学习

一、安装

npm install -g hexo-cli

二、建站

$ hexo init <folder>
$ cd <folder>
$ npm install

三、生成静态页面

hexo generate

四、启动本地服务，进行文章预览调试

hexo server

五、每次部署的步骤

hexo clean
hexo generate
hexo deploy

六、写作

hexo new [layout] <title>
